# QAce Bug Lab

QAce Bug Lab is a Software Quality Assurance portfolio project designed to demonstrate QA processes, test documentation, bug management concepts and modern web application testing.

## Project Goals

- Build a realistic QA testing environment
- Practice functional testing and validation
- Create and maintain test cases
- Document defects and improvements
- Demonstrate QA workflows and best practices

## Current Features

- Welcome Screen
- Login Validation
- Bug Dashboard
- Bug Details Modal
- Severity Filters
- Logout

## Project Structure

```text
app/            Application source code
docs/           Functional documentation
test-cases/     Test case documentation
bug-reports/    Defect reports
screenshots/    Testing evidence
api-tests/      Future API testing artifacts
```
