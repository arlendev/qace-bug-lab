# QAce Bug Lab

QAce Bug Lab is a Software Quality Assurance portfolio project created to demonstrate practical QA activities through a custom-built testing laboratory.

The project combines software development and Quality Assurance practices, providing a controlled environment for test case design, functional testing, regression testing, defect reporting and future test automation initiatives.

## Project Goals

* Build a realistic QA testing environment
* Practice functional testing and validation
* Design and maintain test cases
* Document defects and improvements
* Demonstrate QA workflows and best practices
* Create a public QA portfolio with real testing artifacts
