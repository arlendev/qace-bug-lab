# Functional Requirements

## Entity: Bug

A Bug represents a software defect visualized as an insect card.

### Fields

- id
- name
- bugFamily
- severity
- priority
- status
- description
- symptoms
- recommendedTreatment
- imageUrl

## Bug Families

- Beetle
- Spider
- Ant
- Moth
- Wasp
- Mosquito
- Fly
- Caterpillar

## Severity Levels

- Low
- Medium
- High
- Critical

## Priority Levels

- Low
- Medium
- High

## Status Values

- Open
- In Progress
- Resolved
- Closed
