# QAce Ecosystem

## Vision

QAce is a personal Quality Assurance portfolio and training ecosystem.

The objective is not only to demonstrate software testing knowledge, but also to provide a practical environment where QA concepts, testing techniques and documentation can be applied and showcased.

---

## Ecosystem Structure

### qace.dev

Main entry point.

Purpose:

- Professional presentation
- QA portfolio
- About Me
- Navigation hub

---

### QA Portfolio

Purpose:

- Showcase QA skills
- Present test cases
- Present regression suites
- Present defect reports
- Present testing strategy
- Present future automation roadmap

Target audience:

- Recruiters
- Hiring managers
- Technical interviewers

---

### QAce Bug Lab

Purpose:

- Interactive testing environment
- QA training laboratory
- Demonstrate testing workflows
- Execute manual testing activities
- Support future automation exercises

Current features:

- Login
- Search
- Filter
- Create Bug
- Update Bug
- Delete Bug
- Local Persistence
